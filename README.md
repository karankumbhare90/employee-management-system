
## Run Locally

Clone the project

```bash
  git clone https://github.com/karankumbhare90/employee-management-system.git
```

Go to the project directory

## Frontend
```bash
  cd frontend
```

## Backend
```bash
  cd backend
```

Install dependencies for both frontend as well backend

```bash
  npm install
```

Start the server

## Frontend

```bash
  npm run dev
```

## Backend

```bash
  node server.js
```

Alternative

```bash
  nodemon server.js
```
